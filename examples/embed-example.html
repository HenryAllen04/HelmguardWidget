HelmGuard Widget Embedding Example

Method 1: Auto-Initialize (Recommended)

Add the widget CSS and JavaScript to your page. The widget will automatically initialize when the page loads.

<!-- HelmGuard Widget CSS --> <link rel="stylesheet" href="path/to/helmguard-widget.css"> <!-- HelmGuard Widget JavaScript (auto-initialize) --> <script src="path/to/helmguard-widget.js" data-auto-init="true"></script>

Method 2: Manual Initialization

For more control over when the widget initializes, you can manually call the init function.

<!-- HelmGuard Widget CSS --> <link rel="stylesheet" href="path/to/helmguard-widget.css"> <!-- HelmGuard Widget JavaScript --> <script src="path/to/helmguard-widget.js"></script> <!-- Initialize the widget --> <script> // Initialize when ready HelmGuardWidget.init(); // Or initialize in a specific container // HelmGuardWidget.init('my-widget-container'); </script>

Method 3: Iframe Embedding

For complete isolation, you can embed the widget demo page in an iframe.

<iframe src="https://your-domain.com/helmguard-demo" width="100%" height="700" frameborder="0" style="border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1);"> </iframe>
